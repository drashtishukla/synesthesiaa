🎵 Synesthesia Widget

Embed a crowd-controlled music queue on any website in under 3 lines of code.

Floating Launcher Mode

A draggable bubble appears in the corner of the page. Click it to expand the full music room widget. Drag to reposition. This is the recommended mode for most websites. 

<script src="https://your-app.com/widget.js"></script>
<script>
  Synesthesia.init({
    floating: true,
    width: '400px',
    height: '650px',
    // roomCode: 'ABCD',  // optional: auto-join a room
  });
</script>

Inline Container Mode

Embed the widget inside a specific element on your page. Great for dedicated music pages, dashboards, or sidebars. 

<div id="music-widget"></div>
<script src="https://your-app.com/widget.js"></script>
<script>
  Synesthesia.init({
    container: '#music-widget',
    width: '100%',
    height: '600px',
  });
</script>

📡 Events (postMessage)

The widget sends real-time state updates to the parent page via postMessage. Listen for events to sync your page UI with the music room. 

window.addEventListener('message', function (e) {
  if (e.data && e.data.type === 'synesthesia:state') {
    console.log('Room:', e.data.roomCode);
    console.log('Song:', e.data.currentSong);   // { title, artist }
    console.log('Queue:', e.data.queueLength);
    console.log('Admin:', e.data.isAdmin);
  }
});

🔌 JavaScript API

Control the floating widget programmatically after initialisation.

Synesthesia.open()      // expand the widget panel
Synesthesia.close()     // collapse to the bubble
Synesthesia.toggle()    // toggle open / close
Synesthesia.destroy()   // remove the widget from the page
